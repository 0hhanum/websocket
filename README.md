## Server

- Node.js
- Express
- WebSocket
- Socket.io

```bash
npm i
npm run dev
```

<br>

## Client

- React
- Vite
- Websocket
- Socket.io
- WebRTC
- styled-component
- react-hook-form

<br>

To run the client, you need to run the following commands:

```bash
npm i
npm run dev
```

Or, if you want to build the client-side code for production, run:

```bash
npm i
npm run build
```

**Then, place the created dist directory on server/src/public/dist on your server.**
